# restaurant-pos-frontend

You need to install [bun package manager](https://bun.sh/package-manager) first.

To install dependencies:

```bash
bun install
```

After that, copy `.env.example` into `.env` and modify the `API_BACKEND` to match the backend URL.

## How to Run?

To start a development server:

```bash
bun dev
```

To run for production:

```bash
bun start
```

This project was created using `bun init` in bun v1.2.20. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
